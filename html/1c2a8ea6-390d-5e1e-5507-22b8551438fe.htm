VisualShaderNodeCustom_GetOutputPortName Method

Override this method to define the names of output ports of the associated custom node. The names are used both for the output slots in the editor and as identifiers in the shader code, and are passed in the output_vars array in _GetCode(Array, Array, Int32, Int32).

Defining this method is optional, but recommended. If not overridden, output ports are named as "out" + str(port).
